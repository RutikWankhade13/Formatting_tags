HTML Formatting Elements

Bold text: This text is in bold using <b>.

Strong text: This text is important using <strong>.

Italic text: This text is in italic using <i>.

Emphasized text: This text is emphasized using <em>.

Marked text: This text is highlighted using <mark>.

Small text: This text appears smaller using <small>.

Deleted text: This text has been removed using <del>.

Inserted text: This text has been added using <ins>.

Subscript text: H2O (Water) using <sub>.

Superscript text: X2 (Square) using <sup>.
